# Elephas

Long-term knowledge graph (Chronicle) maintenance. Ingests structured signals from system journals, resolves entity identity, promotes confirmed facts, and generates inferences.

## Repository Structure

- `ocas-elephas/` - Skill package source
- `spec/` - Specification and build documentation

## Installation

This skill package is available on GitHub. To use it, clone this repository and follow the installation instructions in your skill management system.
